# multi_scenario

Cooperative multi-agent reinforcement learning framework targeting four VMAS scenarios — `discovery`, `flocking`, `navigation`, `transport` — across MAPPO / IPPO / MADDPG / IDDPG / ISAC / MASAC. Runs locally or on OVH, emits CSV results, and ships a Streamlit FE for browsing and comparison.

> The folder name `multi_scenario/` is a placeholder while the project lives inside the VMAS repository. It will be renamed and extracted to its own repo at F10.4.

See [`plan.md`](plan.md) for scenario rationale and [`implementation_plan.md`](implementation_plan.md) for the phased implementation roadmap.

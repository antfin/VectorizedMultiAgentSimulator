# Getting started

Three things you'll want to do first:

1. **[Install](installation.md)** — environment + dependencies.
2. **Pick a config** under `experiments/discovery/baseline/configs/` or `experiments/discovery/lero/configs/`.
3. **[Submit](submitting_experiments.md)** — via the Streamlit Submit page (canonical) or the `multi-scenario run` CLI.

For a worked example, see [example_config.yaml](example_config.yaml).

# Results analysis

After a run completes, results land under
`<storage.path>/<run_id>__<timestamp>/`. Three views:

- **[Streamlit dashboard](streamlit_dashboard.md)** — browse one or many runs interactively.
- **[CSV aggregation](csv_aggregation.md)** — `multi-scenario consolidate` produces a CSV across all runs in a tree.
- The [Reproducibility](../reproducibility/index.md) section pins specific science results.

See also: [Frontend → Run detail page](../frontend/run_detail_page.md) for one-run drill-down.

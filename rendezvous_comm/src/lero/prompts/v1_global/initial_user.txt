## Scenario: VMAS Discovery

The "Discovery" scenario is a cooperative multi-agent environment built in VMAS (Vectorized Multi-Agent Simulator). It runs hundreds of parallel environments simultaneously using PyTorch tensors.

### Task Description

- $n_agents agents must cover $n_targets targets in a 2D bounded arena ([-1, 1] x [-1, 1]).
- A target is "covered" when at least $agents_per_target agents are within $covering_range distance of it.
- Once covered, a target is moved outside the arena (it does not respawn). The episode succeeds when ALL targets have been covered at some point.
- Episodes last up to $max_steps steps.

### Agent Capabilities

- **Movement**: Continuous 2D force actions.
- **Target LiDAR**: $n_lidar_rays_entities rays, range $lidar_range — detects nearby targets.
- **Agent LiDAR**: $n_lidar_rays_agents rays, range $lidar_range — detects nearby agents. $agent_lidar_description

Previous experiments in this project:
- **ER1 (No Communication)**: Agents coordinate using only LiDAR observations. Agents with LiDAR can detect targets and other agents but have no explicit communication. This is the baseline. Key finding: k=2 (2 agents per target) is very hard — success rates of 3-32%.
- **ER2 (Learned Communication)**: Agents have continuous communication channels (dim_c floats per step). Messages are proximity-gated (only nearby agents hear each other). Agents must learn what to communicate. Key finding: communication helps but agents struggle to learn useful protocols.
- **ER3 (Graph Neural Network)**: Instead of explicit messages, a GNN aggregates neighbor observations via attention. Achieved 71% success on k=2.

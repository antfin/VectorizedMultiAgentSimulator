## Examples

Below are two example reward functions from related MPE Simple Spread tasks. They illustrate vectorized patterns, credit assignment (individual vs team), and shaping terms. Use them as references, not templates — design the function best suited to THIS task.

### Example A — dense shaping

```python
def compute_reward(scenario_state: dict) -> torch.Tensor:
    # Each agent penalized by distance to its nearest landmark
    dists = torch.cdist(
        scenario_state["agent_pos"].unsqueeze(1),
        scenario_state["targets_pos"],
    ).squeeze(1)                                  # [B, n_targets]
    nearest = dists.min(dim=-1).values            # [B]
    # Team-wise landmark coverage: sum of min agent-landmark distance
    all_dists = scenario_state["agents_targets_dists"]        # [B, n_agents, n_targets]
    team_penalty = all_dists.min(dim=1).values.sum(dim=-1)    # [B]
    return -nearest - 0.1 * team_penalty + scenario_state["collision_rew"]
```

### Example B — coverage bonus + collision penalty

```python
def compute_reward(scenario_state: dict) -> torch.Tensor:
    covered = scenario_state["covered_targets"].float().sum(dim=-1)   # [B]
    all_covered = scenario_state["covered_targets"].all(dim=-1).float()
    reward = 5.0 * covered + 20.0 * all_covered
    reward = reward + scenario_state["collision_rew"]
    reward = reward + scenario_state["time_penalty"]
    return reward
```

## Example

A minimal vectorized observation function showing the expected output format. It returns one trivial feature (nearest LiDAR-target distance) — design something more useful for THIS task.

```python
def enhance_observation(scenario_state: dict) -> torch.Tensor:
    lidar_t = scenario_state["lidar_targets"]    # [B, n_rays]
    nearest_target_dist, _ = lidar_t.min(dim=-1) # [B]
    return nearest_target_dist.unsqueeze(-1)     # [B, 1]
```
